There are 3 types of list:

definition list

    which show terms and definations

unordered lists

    which show bullets

ordered lists

    which show numbers or letters

unordered lists can have a type attribute
which can be applied to the UL or the LI tag.
The three values allowed are:

  • type = "Square"
  • type = "disc"
  • type = "circle"
